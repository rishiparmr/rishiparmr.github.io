## üêõ Troubleshooting

**Dev server won't start:**
```bash
rm -rf node_modules package-lock.json
npm install
npm run dev
```

**Build fails:**
- Check for console errors
- Ensure all imports are correct
- Verify Vue syntax in all components

**Styles not applying:**
- Check CSS variable names
- Verify import in main.js
- Clear browser cache

**Links not working:**
- Update all `href="#"` with real URLs
- Test smooth scroll in production build
